# Sample Queries in Thai

Here are some example natural language queries in Thai that ClawGraph can handle:

## Basic Queries
1. **project อะไรที่ deadline เดือนนี้**  
   - Find projects with deadlines in the current month

2. **client ไทยทั้งหมด**  
   - List all Thai clients

3. **expense รวมของ project X**  
   - Total expenses for project X

## Relationship Queries
4. **project ที่เกี่ยวข้องกับ client ABC**  
   - Projects related to client ABC

5. **task ที่มี deadline ใกล้ที่สุด**  
   - Tasks with the nearest deadlines

6. **expense ที่มากกว่า 5000 บาท ของ client ไทย**  
   - Expenses over 5000 baht from Thai clients

## Complex Queries
7. **สรุป project ที่กำลังจะ deadline และเกี่ยวข้องกับ client ไทย**  
   - Summarize projects nearing deadline related to Thai clients

8. **แสดงความสัมพันธ์ระหว่าง project X และ expense ของมัน**  
   - Show relationships between project X and its expenses

9. **client ที่มี project มากที่สุด**  
   - Clients with the most projects

10. **task ที่ยังไม่เสร็จของ project ที่ deadline เดือนหน้า**  
    - Incomplete tasks for projects due next month

## Advanced Queries
11. **project คล้ายกับ project Y**  
    - Projects similar to project Y (using vector similarity)

12. **path จาก client ABC ไปยัง expense ของ project X**  
    - Path from client ABC to expenses of project X

These queries demonstrate the natural language capabilities of ClawGraph for querying the knowledge graph.
